## Setup envs
- copy `.env.example` to `.env`
- change vars to correct ones

## Running with docker

```
docker-compose up
```

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# e2e tests
$ pnpm run test:e2e
```
